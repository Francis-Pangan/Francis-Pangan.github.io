Tell a story about how you got blocked on a simple problem, and how you solved the problem in an elegant way.

I was trying to complete the FizzBuzz challenge and was struggling with how to write the IF ELSE statements. It was hard to get my head around the decision tree at first.

Then I thought about how similar IF ELSE decision trees are to a flow chart, and that was the solution! I wrote down how I would like my code to flow, and looked up what symbols to use with my code, and was able to map out how the IF, ELSE IF, ELSE statements would work. I wrote pseudocode and used console.log quite a bit to tweak my code, but it in end I got everything tickety-boo.

I felt pretty good about the process, and learned that tackling big problems are easier once you step back and break down the problem. I think I would want to develop more problem solving techniques as I go to minimize stress and frustration with code.
